=================
Py3o UBL Invoices
=================

If you use the py3o reporting engine (module *report_py3o* from the `reporting-engine <https://github.com/OCA/reporting-engine>`_ OCA project) to generate your PDF invoices and you want these PDF documents to have an embedded UBL XML file, this module is for you !

Configuration
=============

No configuration is needed.
